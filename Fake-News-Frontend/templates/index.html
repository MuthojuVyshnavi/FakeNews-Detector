About TruthLens
How it works and what powers it.

How It Works

1

Input text or URL

Paste any news article, social media post, or provide a URL. You can also use voice input.

2

ML model analyzes

A trained machine learning model (TF-IDF + classifier) processes the text, extracting linguistic patterns associated with fake or real news.

3

Results explained

You get a verdict, confidence score, sentiment analysis, keyword extraction, and highlighted suspicious phrases — not just a label.

What to trust

  • High confidence (85%+) results are generally reliable
  • Low confidence (below 65%) means the model is uncertain — treat with caution
  • Always cross-check with trusted news sources
  • The model may not catch sophisticated misinformation

Tech stack

  • Frontend: Vanilla HTML, CSS, JavaScript
  • Backend: Python Flask REST API
  • ML: scikit-learn (TF-IDF + classifier)
  • Database: SQLite via Python
  • Auth: SHA-256 password hashing

Limitations

TruthLens is a demonstration project. The ML model was trained on a specific dataset and may not generalize perfectly to all types of content. It works best on English-language news articles. Do not use this as the sole basis for sharing or not sharing content — always apply critical thinking and consult multiple sources.
